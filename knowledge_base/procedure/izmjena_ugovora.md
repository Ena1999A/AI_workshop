# Izmjena ugovora

## Opis
Izmjena ugovora obuhvaća svaku promjenu izvornih ugovornih uvjeta nakon aktivacije ugovora.

## Vrste izmjena
- promjena plana otplate
- promjena datuma dospijeća rate
- produljenje trajanja ugovora
- restrukturiranje zbog prijevremene otplate
- privremeno smanjenje obroka

## Koraci postupka
1. Zahtjev stranke evidentira se u CRM sustavu.
2. Agent provjerava zadovoljava li zahtjev uvjete prihvatljivosti.
3. Izračunava se financijski učinak predložene izmjene.
4. Generira se ažurirani plan otplate.
5. Priprema se aneks ugovora.
6. Aneks se potpisuje elektronički ili drugim odobrenim putem.
7. Parametri ugovora ažuriraju se u sustavu.

## Napomena
Sve izmjene moraju biti sljedive i evidentirane u povijesti ugovora.
